Detailed Description

usage of a particular UART, taking care of hardware-level calls, OS-level calls, and passing up statuses. OS support is selected at compile time based on whether the THREADED macro is defined

Parameters
arrReceiveThe byte array which the received data is to be written into
numBytesThe number of bytes to be received
Returns
True if the transfer succeeded, otherwise false
Note
Some reasons why false may be returned include:
  1. Incomplete initialization of the driver
  2. UartInterface returns an error upon requesting a transfer
  3. OS block time is exceeded
