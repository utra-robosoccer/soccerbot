Macro name: dfile
Macro description: Doxygen comment for the top of a file
Macro expansion:
/**
 * @file
 * @author TODO -- your name here
 * @brief TODO -- briefly describe this file if necessary
 *
 * @defgroup TODO -- name of this module
 * @brief TODO -- brief description of this module
 * @ingroup TODO -- parent module (if it exists)
 * @{
 */




Macro name: dfunc
Macro description: Generic Doxygen comment for a function
Macro expansion:
/**
 * @brief
 * @details
 * @param
 * @retval
 */
